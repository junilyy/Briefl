# BRIEFL Frontend

BRIEFL React frontend workspace.

## Tech Stack

- React
- TypeScript
- Vite
- ESLint

## Local Development

```bash
npm install
npm run dev
```

Default dev server:

```text
http://localhost:5173
```

## Verification

```bash
npm run lint
npm run build
```

## Notes

- Google Apps Script feedback submission is intentionally not implemented yet.
- Backend feedback APIs should not be added.
- The root `index.html` file is a local reference artifact and must not be committed.
